HOW TO INSTALL:

This project has 2 Parts. 
PPT Remote Client and PPT Remote Server.
Client module is for the mobile.
Server module is for the desktop/laptop.
send and install the PPTRemote_Clinet.jar in your java enabled mobile and just run the PPTRemote_Server.jar in your system.
