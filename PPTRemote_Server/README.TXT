PPT Remote is a mobile based presentation controller/remote. 
-------------------------------------

-------------------------------------
HOW TO RUN :

- go to the 'dist' folder and execute the PPTRemote_Server.jar file
- or just run the 'RUN.BAT' file
-------------------------------------
TROUBLESHOOT:

it the program cannot detect the bluetooth stack then check whether the bluetooth device connectedand recognised by the os.
if problem still exists then copy the win32lib.dll file present in the system_dll folder to the bin folders in the java directory.

that is in 
1. C:\Program Files\Java\jdk1.6.0_07\bin
2. C:\Program Files\Java\jdk1.6.0_07\jre\bin
3. C:\Program Files\Java\jre6\bin
